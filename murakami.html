村上さんが作るものだけを、上から順番に作るページです

むずかしい説明は飛ばします。このページの順番どおりに作ると、最後にLINEへ写真と文を送って、Instagram投稿テストができます。

Facebookも新しく作る 名前はデモ用 パスワードはここに書かない 困ったら止める

まず大きく見る: 作るものは4つ、最後に登録

今日やることはこれだけです。上から順番に進みます。

1Facebook個人村上さん本人のアカウント
2Facebookページ楽スタデモ村上店
3Instagram投稿が出る場所
4公式LINE写真と文を送る場所
5BANTEX確認と楽スタ登録LINEとInstagramをつなぐ
最後は、公式LINEに「写真」と「文」を送って、楽スタに投稿案が出るか見ます。

始める前に用意するもの

  • 村上さんが使えるメールアドレス
  • SMSを受け取れるスマホ
  • Facebook用に決めたパスワード
  • Instagram用に決めたパスワード
  • LINEにログインできるアカウント
  • テスト投稿に使う写真1枚
  • 楽スタ管理画面のログイン情報
パスワード、SMSコード、LINEのSecret、tokenは、このページやチャットに書かないでください。画面に直接入力します。

1. Facebook個人を作る

最初に、村上さん本人のFacebookアカウントを作ります。

やること: Facebookを開き、新しいアカウントを作ります。
名前: 村上さんの本名を入れます。店名や架空名にしません。
止まる画面: SMS、本人確認、2段階認証が出たら、村上さん本人が画面で入力します。
Facebook個人の名前は、最後のInstagram投稿には出ません。

2. Facebookページを作る

次に、デモ店舗用のFacebookページを作ります。

場所入れるもの注意
ページ名楽スタデモ村上店「デモ」を必ず入れる
カテゴリサービス、店舗、ローカルサービスなど近いもの迷ったら近いものでよい
住所空欄、またはBANTEX管理の情報実在店舗の住所は入れない
画像仮画像またはロゴ本物の店舗ロゴは使わない

3. Instagramを作る

Instagramは、最後に投稿が表示される場所です。

場所入れるもの注意
表示名楽スタデモ村上店Facebookページと同じ
ユーザー名rakusta_demo_murakami使えなければ末尾に01などを付ける
アカウント種類プロアカウント / ビジネスあとで必ず切り替える
リンク先Facebookページ「楽スタデモ村上店」前の手順で作ったページを選ぶ
Instagramを作る
メール、名前、ユーザー名を入れて作ります。
プロアカウントに切り替える
画面の案内に沿って、ビジネス用にします。
Facebookページとつなぐ
「楽スタデモ村上店」を選びます。

4. 公式LINEを作る

公式LINEは、写真と文を送る場所です。

場所入れるもの注意
公式LINE名楽スタデモ村上店Instagramと同じ名前
Messaging API有効にするここを有効にしないと楽スタに届かない
Webhook URLhttps://kokotomo-toc-bot.onrender.com/callbackUse webhookをONにしてVerifyする
Channel Access Token楽スタ管理画面に貼るメモに残さない
Channel Secret楽スタ管理画面に貼るメモに残さない

5. BANTEXでInstagramを使えるようにする

ここはBANTEX管理者ログインが必要です。作ったInstagramをBANTEXから見える状態にします。

やること: Facebookページ「楽スタデモ村上店」をBANTEXに追加します。
やること: Instagram「rakusta_demo_murakami」をBANTEXに追加します。
確認: BANTEX投稿Botから、このInstagramが見えるか確認します。
ここで分からなければ止めてください。Instagram User IDはBANTEX側で確認してから、次の楽スタ管理画面に入れます。

6. 楽スタ管理画面に登録する

ここで、作ったLINEとInstagramを楽スタに登録します。

場所入れるもの注意
店舗名楽スタデモ村上店「デモ」を残す
LINE情報LINE DevelopersのTokenとSecret貼ったら保存。メモに残さない
Instagram名rakusta_demo_murakami実際に取れた名前を入れる
Instagram User IDBANTEX側で確認したID分からなければここで止める
Token運用方式BANTEX共通店舗ごとのtokenは作らない

7. LINEからテストする

ここまでできたら、最後にLINEでテストします。

公式LINEを友だち追加
「楽スタデモ村上店」の公式LINEを開きます。
写真を1枚送る
テスト用の写真を送ります。
文を送る
「テスト投稿です。楽スタデモ村上店の投稿確認です。」と送ります。
楽スタのダッシュボードを見る
投稿案ができているか確認します。
必要ならInstagramへ投稿
投稿先がデモInstagramになっていることを見てから投稿します。
投稿案が出たら、ひとまず成功です。
