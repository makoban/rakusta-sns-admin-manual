新しい店舗アカウントで、LINEからInstagram投稿できる状態まで作る

このページは、遠方の社内管理者が一人で作業できるようにした操作マニュアルです。実パスワードやtokenはこのHTMLには書かず、別の安全な方法で共有してください。

店舗側のLINE運用は今まで通り Instagram tokenは店舗ごとに発行しない BANTEX共通token方式 スクショ多め

人間が作業すると何分かかるか

初心者が初回でやるなら、全体で 2時間から3時間 を見てください。Meta/Facebookの認証やSMSで止まると、その場で終わらない場合もあります。

最短45分から75分。Facebook個人、Instagram、LINE公式がすでにあり、ログインも通る場合。
通常90分から150分。メール、Facebook個人、Facebookページ、Instagram、LINE公式を順に作る場合。
初心者初回120分から180分。迷い・認証・画面差分・Metaログイン確認を含む現実的な時間。
止まるケースFacebook個人の本人確認、Instagramログイン制限、SMS未着、Metaの一時ロック。数時間から数日待つことがあります。

最初に用意するもの

店舗側で必要

  • 店舗名、住所、電話番号
  • 店舗メールアドレス
  • 店舗責任者の実在Facebook個人アカウント
  • 店舗Instagram用ユーザー名候補を3つ
  • プロフィール画像、ロゴ、カバー画像
  • テスト投稿用の写真1枚と短いメモ

BANTEX側で必要

  • 楽スタ管理画面の管理者ログイン
  • LINE Official Account Managerのログイン
  • LINE Developersの操作権限
  • Meta Business Suite / Business設定の管理権限
  • BANTEX投稿Botを扱えるMeta管理者権限
  • Render / 本番環境は通常触らない。token更新時だけBANTEX側で扱う
Facebook個人は架空名で作らない。 店舗責任者または社内管理者の実在個人で作ります。架空アカウントは後からMeta連携や本人確認で止まりやすくなります。

1. 店舗メールアドレスを用意

新規で用意する場合はGoogleアカウントでも構いません。法人ドメインメールがあるならそちらが望ましいです。

店舗メールはInstagram、LINE、Facebook通知の受け皿です。後から誰が見られるか分からなくならないよう、管理表に「用途」と「管理者」を残します。

2. Facebook個人アカウントを作成または確認

Facebook個人は店舗ページを作るための入口です。店舗名ではなく、実在する個人で作ります。

SMS認証、本人確認、2FAが出たら、本人または権限を持つ管理者が画面上で入力します。コードをチャットに貼らないでください。

3. Facebook店舗ページを作成

Facebookにログインした状態で、店舗ページ作成に進みます。

  • ページ名は店舗名にする
  • カテゴリ、住所、電話番号、公式URLを入れる
  • プロフィール画像とカバー画像を設定する
  • ページURLとPage IDを控える

4. 店舗Instagramを作成し、ビジネス化

Instagramを作成する。
プロフィールから「プロアカウントに切り替える」を選ぶ。
カテゴリを選び、ビジネスアカウントにする。
Facebook店舗ページとInstagramをリンクする。
ここまで終わると、Meta APIで投稿できる土台ができます。Instagramだけ作ってもAPI投稿はできません。Facebookページとのリンクが必要です。

6. BANTEX側Meta Businessへ追加

店舗のFacebookページとInstagramをBANTEX側Business Portfolioに追加し、BANTEX投稿Botから見える状態にします。

  • Business PortfolioにFacebook店舗ページを追加
  • 同じBusiness Portfolioに店舗Instagramを追加
  • FacebookページとInstagramがリンク済みか確認
  • BANTEX投稿Botの権限で対象Instagramが見えるか確認
  • 既存店舗のページ/Instagram権限を外さない
今後の新店舗では、店舗ごとにGraph API Explorerでtokenを作りません。Meta側で投稿先資産をBANTEX管理下に入れ、楽スタ側は instagram_token_source=global にします。

7. 楽スタ管理画面に登録

楽スタ管理画面にログインし、店舗を検索してLINE/Instagram/投稿設定を保存します。

8. LINEからInstagram投稿まで確認

店舗LINE公式に、テスト写真1枚と「テスト投稿です」などのメモを送る。
楽スタのSNSダッシュボードに投稿案が出るか確認する。
投稿文、画像、予約時間を確認する。
Instagram APIヘルスチェックを行う。
必要なら1回だけInstagramへ実投稿する。

ヘルスチェックコマンド

作業者がターミナルを使える場合は、BANTEX側で以下を実行します。

uv run --with requests --with sqlalchemy --with psycopg2-binary \
  python scripts/check_instagram_health.py --franchise-id <NEW_FRANCHISE_ID>
OK条件は、graph_status=ok、profile取得200、content_publishing_limit 200です。

失敗時の見方

症状原因の層確認する場所
LINEから送ったのに投稿案が出ないLINE Webhook / Channel tokenLINE Developers、楽スタ管理画面のLINE欄
投稿案は出るがInstagram投稿できないInstagram権限 / Meta資産 / User IDMeta Business、楽スタInstagram欄、ヘルスチェック
Invalid OAuth access token古い個別tokenToken運用方式をglobalにする
Unsupported get requestBANTEX共通tokenからそのInstagramが見えていないMeta Businessにページ/Instagramを追加
Application does not have permissionBANTEX投稿Botの権限不足Meta OAuth権限、Instagramリンク、ページ選択
Facebook/Instagramが途中で止まる本人確認、SMS、2FA、Meta制限本人または管理者がブラウザ上で対応
新規Facebook個人アカウントは、作成直後にページ作成やBusiness連携を連続で行うと制限されることがあります。急がず、本人確認が出たら画面通りに処理してください。

配布時の注意

  • GitHub Pagesで配る場合、このページ自体はURLを知っている人が見られます。
  • 実パスワード、token、App Secret、LINE secretは絶対に入れない。
  • このHTMLには noindex,nofollow を入れていますが、アクセス制限ではありません。
  • 認証情報は別ルートで、権限のある社内管理者にだけ共有する。
